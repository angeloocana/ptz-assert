> ptz-assert is a wrapper for npm assert
>
> Be welcome to learn, help and play!!!

## Install

Install with npm:

```bash
    $ npm install ptz-assert --save
```

or clone from github:

```bash
    $ git clone https://github.com/angeloocana/ptz-assert.git
```

## Setup

```bash
    $ npm install 
    $ typings install
```

## Test

```bash
    $ npm test
```
